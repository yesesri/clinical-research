FDA_IVD_Summary - PDF file 

```
This study guide provides a comprehensive review of the regulatory requirements for medical devices and in vitro diagnostics (IVDs) in the United States.
It focuses on the criteria for submitting new premarket notifications (510(k)) for device modifications, the transition from the Quality System Regulation (QSR)
to the Quality Management System Regulation (QMSR), and the classification of IVD products.
```
